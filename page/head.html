head

La balise < head > contient l'entête du document html, c'est à dire toutes les informations non visibles par l'internaute. C'est une balise qu'il ne faut pas oublier. Elle fait partie des 4 balises Html qui doivent être présentes dans tous les documents HTML

Exemple:
< head > < title>< / title > < meta charset="utf-8"> < / head >

la balise < head >est tres importante dans une page html elle renseigne entre autre des informations concernant le refererencement.
